<ascii-image> demo

<ascii-image> is like the <img> tag, but it renders the image as ASCII characters.

It accepts a width attribute so you can define the number of columns:

<ascii-image src="grumpy.jpg" columns="60"></ascii-image>

Don't use this to save data because you're still loading the original image.

If you don't specify the number of columns, it will default to 80 columns.
